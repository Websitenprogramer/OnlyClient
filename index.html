Dein Minecraft-
Client, dein Stil

OnlyClient verwaltet Instanzen, installiert Mods von Modrinth, bringt Capes & Emotes mit — und lässt sich komplett anpassen.

OnlyClient installieren OnlyClient-Setup.exe · Windows Only+ Shop o+-Abzeichen im Spiel

Installer fehlt — bitte Seite neu deployen nach npm run build:website

Updates kommen automatisch — nach der Installation musst du nichts mehr manuell laden.

Was OnlyClient kann

🧩

Mods & Modpacks

Modrinth durchsuchen, per Drag & Drop installieren, Fabric/Quilt/NeoForge.

🎨

Design & Hintergrund

Farben, Leuchteffekte, Partikel, eigenes Hintergrundbild — Deutsch oder Englisch.

👤

Profil & Capes

Skin hochladen, OnlyClient-Capes, EmoteCraft im Spiel.

▶

Spielen

Instanzen verwalten, Server speichern, separates Log-Fenster beim Start.

Installation

  1. OnlyClient-Setup.exe herunterladen und ausführen.
  2. Installation durchklicken — Desktop-Verknüpfung „OnlyClient“ erscheint.
  3. OnlyClient öffnen, mit Microsoft anmelden, Instanz erstellen, spielen.

Benötigt Minecraft Java Edition (Microsoft-Konto). Kein offizielles Mojang-Produkt.
